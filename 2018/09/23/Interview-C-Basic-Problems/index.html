数组指针

有n个人围成一圈，顺序排号。从第一个人开始报数（从1到5报数），凡报到5的人退出圈子，后面的人接着从1报数，直到只剩最后一人，问最后留下的是原来第几号。提示：用数组实现，数组元素置为0表示该元素退出圈子。要求引入指针变量完成。

  • 32位编译器

    char ：1个字节
    char*（即指针变量）: 4个字节（32位的寻址空间是2^32, 即32个bit，也就是4个字节。同理64位编译器）
    short int : 2个字节int： 4个字节

    float: 4个字节
    double: 8个字节
